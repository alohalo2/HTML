figma 단축키 정리

1. 속성 복사 : ctrl + alt + c -> 속성 복사할 객체 선택 (고급설정은 복사 불가)

2. 최하위 오브젝트 선택: ctrl + 마우스 오른쪽 클릭 (잠긴 레이어도 표시되어 그룹화가 복잡하게 되어 있을 경우 원하는 레이어를 더 손쉽게 찾을 수 있음)

3. 섬네일 설정하기: 섬네일로 설정할 프레임 선택 -> 마우스 오른쪽 클릭 -> set as thumnail 실행

4. 오브젝트 간격 확인하기: 오브젝트 선택  -> alt / alt 누른채  + ctrl하면 최하위 오브젝트와의 간격 확인 가능

5. 부모-자식 레이어 쉽게 전환하기(여러 레이어 선택)
부모 레이어 선택 + Enter / 자식 레이어 선택 + Shift + Enter

6. 모든 레이어 닫기: alt + L 

7. 여러 레이어 열기: ctrl+ A -> Enter
